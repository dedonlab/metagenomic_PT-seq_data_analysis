These are demo raw reads and trimmed reads of paired metagenomic PT-seq for donor 5 and demo reference genomes.  
Download the reads and put them in the work/demo/ folder.  
To deplete human sequence contamination, please download the hg19_main_mask_ribo_animal_allplant_allfungus.fa.gz file at https://zenodo.org/records/1208052 and place it in the work/demo folder with the demo reads.
Put the tar.gz file in the work/demo/ folder and extract the files.  
User can also start with the trimmed reads.
